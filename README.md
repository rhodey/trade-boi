# trade-boi
dat boi trading on gdax.

## Build
```
$ sbt assembly
```

## Config
```
$ cp example-scam.properties scam.properties
```

## Run
```
$ java -jar target/scala-2.12/trade-boi-assembly-0.3.0.jar
```

## Development
```
$ build-proto.sh
```
