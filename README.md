# coinbase

messing around with coinbase trading.

## Build
```
$ mvn package
```

## Config
```
$ cp example-scam.properties scam.properties
```

## Run
```
$ java -jar target/coinbase-0.1.jar
```

## Development
```
$ build-proto.sh
```
